# OGH Style Guide

This style guide provides standards and documentation for building and maintaining the OGH Wix (non-Studio) website, including components, pages, and design decisions.

## Table of Contents

- [Colors & Typography](docs/colors-and-typography.md)
- [Components](docs/components.md)
- [Pages](docs/pages.md)
- [Best Practices](docs/best-practices.md)

## Overview

This repository documents the design system and component patterns used on the OGH Wix website. It is intended to ensure consistency across all pages and elements built within the Wix editor.

### Platform

- **Platform:** Wix (non-Studio / Classic Editor)
- **Site type:** Standard Wix website

### How to Use This Guide

1. Refer to [Colors & Typography](docs/colors-and-typography.md) when applying brand colors, fonts, or text styles.
2. Use the [Components](docs/components.md) guide when adding or editing reusable elements such as buttons, galleries, and forms.
3. Follow the [Pages](docs/pages.md) guide for standard page layouts and structure.
4. Review [Best Practices](docs/best-practices.md) before publishing or making major changes to the site.
