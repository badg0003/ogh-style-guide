# Colors & Typography

This document defines the brand color palette and typography standards for the OGH Wix website.

---

## Color Palette

Wix stores colors in **Site Colors** under the Design panel. The table below defines each color's role. When applying color in the Wix editor, always select the corresponding site color rather than a custom hex value to keep the palette consistent.

| Role | Name | Hex | Usage |
|------|------|-----|-------|
| Primary | Brand Primary | `#1A1A2E` | Main headings, primary CTA buttons |
| Secondary | Brand Secondary | `#16213E` | Sub-headings, footer background |
| Accent | Brand Accent | `#E94560` | Hover states, highlights, links |
| Background Light | Background 1 | `#F5F5F5` | Page backgrounds, card backgrounds |
| Background Dark | Background 2 | `#0F3460` | Dark section backgrounds |
| Text Dark | Text Primary | `#212121` | Body text, labels |
| Text Light | Text Secondary | `#FFFFFF` | Text on dark backgrounds |
| Border | Neutral | `#CCCCCC` | Dividers, input borders |

### Applying Colors in Wix

1. Open the **Design** panel for any element.
2. Click the color swatch.
3. Switch to the **Theme** tab and select the appropriate site color by name.
4. Avoid using the **Custom** color picker unless the color is not in the palette.

## Typography

### Font Families

| Role | Font | Weight | Notes |
|------|------|--------|-------|
| Heading 1 (H1) | Playfair Display | Bold (700) | Page titles only |
| Heading 2 (H2) | Playfair Display | Semi-Bold (600) | Section headings |
| Heading 3 (H3) | Montserrat | Semi-Bold (600) | Sub-section headings |
| Heading 4–6 | Montserrat | Medium (500) | Card titles, sidebar headings |
| Body | Montserrat | Regular (400) | All paragraph text |
| Caption / Label | Montserrat | Regular (400) | Captions, metadata, helper text |
| Button | Montserrat | Bold (700) | All button labels |

### Font Sizes

| Style | Desktop | Mobile |
|-------|---------|--------|
| H1 | 52 px | 32 px |
| H2 | 40 px | 28 px |
| H3 | 30 px | 22 px |
| H4 | 22 px | 18 px |
| Body | 16 px | 15 px |
| Caption | 13 px | 12 px |

### Line Height & Spacing

- **Line height:** 1.6 for body text; 1.2 for headings.
- **Letter spacing:** 0 for body; `0.05em` for button labels and all-caps text.
- **Paragraph spacing:** 16 px below each paragraph.

### Applying Text Styles in Wix

1. Select a text element in the editor.
2. Open **Text Settings** and choose the matching **Heading** or **Paragraph** preset.
3. Do not override font sizes or weights inline unless creating a deliberate exception that is documented here.

---

## Accessibility Notes

- Maintain a minimum contrast ratio of **4.5:1** for body text against its background (WCAG AA).
- Maintain a minimum contrast ratio of **3:1** for large text (18 px+ or 14 px+ bold) against its background.
- Never convey meaning through color alone; always pair color with a text label or icon.
